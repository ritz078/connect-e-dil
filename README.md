Features
--------

1. Send yellow or red roses to friends or valentine.
2. Shoutbox where everyone can post literally anything :stuck_out_tongue_winking_eye:


Getting Started
---------------

* Clone the repo.
* Install ```NodeJS``` ```Grunt``` and ```bower```
* Run ```npm install && bower install```
* Run ```grunt serve```
